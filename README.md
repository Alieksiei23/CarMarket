# 🚗 Carmarket

**Carmarket** is a universal platform that connects showrooms, sellers, and buyers. 
The project enables participants to interact conveniently, post and browse listings, manage their vehicle inventory, 
and track deals in real-time.

## 📝 Description

Carmarket is designed to streamline the process of buying and selling vehicles among different market participants. 
The platform offers an intuitive interface and powerful features for:

- **Showroom** — manage vehicle inventory and tracking special offers
- **Seller** — publish vehicle offers and partner with dealerships  
- **Buyers** — search, compare, and purchase vehicles

## 🚀 Key Features

- 🔍 Convenient car search using filtering, searching and ordering
- 🧾 Post and moderate listings
- 📦 Manage inventory for dealerships and suppliers
- 📈 Analytics for clients
